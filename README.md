# mdn

> Man pages for web APIs using [MDN][mdn]

![screenshot](screenshot.png)

## Install

```sh
$ npm i @rafaelrinaldi/mdn -g
```

## Usage

```sh
$ mdn array.of
$ mdn background-image --language=css
$ # Will open the MDN page on user's default browser
$ mdn function.bind --open
```
